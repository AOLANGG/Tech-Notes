## 字符串函数

### strlen

`sizeof和strlen的区别`

| 特性       | `sizeof`                     | `strlen`                        |
| ---------- | ---------------------------- | ------------------------------- |
| 性质       | 运算符（编译期计算）         | 库函数（运行期计算）            |
| 作用对象   | 任意数据类型 / 变量 / 表达式 | 以`'\0'`结尾的字符串（`char*`） |
| 计算内容   | 内存总字节数（包含所有部分） | 有效字符数（不包含`'\0'`）      |
| 依赖`'\0'` | 不依赖                       | 必须依赖（否则行为未定义）      |
| 头文件     | 不需要                       | 需要`#include <string.h>`       |

`模拟实现strlen`

```c++
size_t myStrlen(const char* str) {
	size_t len = 0;
	if (str) {
		while (*str) {
			len++;
			str++;
		}
	}
	return len;
}
```
